CSS

Learning CSS this time around has been a whole other animal entirely. While I recognized certain elements in the stylesheet, I never knew the names nor how to write the CSS from scratch. this will be a test of knowledge and memory if I can regurgitate everything I've been learning.

With CSS, I should be able to do the following:

  1. Work with text

    • installing fonts
    • Sizing Text
    • Spacing Text
    • Decorating text

  2. Understand styling selectors

    • tags
    • parent > child relationships
    • ancestor > descendants relationship
    • selectors
    • :pseudo-selectors
    • [attribute-selectors]

  3. Box Model

    • Height, width
    • padding
    • borders
    • margin

  4. Advanced stuff

    • Layout and positioning
    • Flexbox
    • Media queries
